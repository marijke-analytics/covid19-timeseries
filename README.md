# covid19-timeseries
COVID‑19 Time Series Analysis – Global Trends, Moving Averages &amp; Bubble Map Visualization

# 🌍 COVID‑19 Zeitreihen & Weltkarte (Python, Cartopy)

## 📌 Überblick
Dieses Projekt analysiert globale COVID‑19 Trends anhand täglicher Fallzahlen, 7‑Tage‑Mittelwerte und geografischer Visualisierungen.  
Ziel ist es, Infektionswellen sichtbar zu machen und die am stärksten betroffenen Länder auf einer Weltkarte darzustellen.

---

## 🎯 Ziele
- Berechnung täglicher Fallzahlen  
- Erstellung von 7‑Tage‑Mittelwerten  
- Identifikation globaler Infektionswellen  
- Visualisierung der Top‑30 Länder auf einer Bubble‑Map  
- Vergleich von Trends zwischen Ländern  

---

## 🧠 Methoden & Tools
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Cartopy (Geodaten & Weltkarte)  

---

## 📊 Visualisierungen
- Zeitreihenplots  
- 7‑Tage‑Mittelwerte  
- Peaks & Wellen  
- Bubble‑Map der Top‑30 Länder  

Alle Visualisierungen liegen im Ordner `visuals/`.

---

## 📁 Projektstruktur
project/
├── data/          # COVID-Daten (z. B. OWID)
├── notebooks/     # Jupyter Notebooks (EDA, Zeitreihen, Karten)
├── src/           # Python-Skripte
├── visuals/       # Plots & Karten
└── docs/          # Dokumentation


---

## 🚀 Ergebnisse
- Die 7‑Tage‑Mittelwerte glätten die täglichen Schwankungen und zeigen klare Infektionswellen.  
- Die Bubble‑Map zeigt die globalen Hotspots und ermöglicht einen schnellen Überblick über die am stärksten betroffenen Regionen.
